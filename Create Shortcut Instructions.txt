To create a desktop shortcut for the Split Screen App:

1. Right-click on your desktop
2. Select New > Shortcut
3. In the location field, paste this command:
   "C:\Users\Kieran Jackson\AppData\Local\Programs\Python\Python311\python.exe" "C:\Users\Kieran Jackson\screen_split_app\screen_app_debug.pyw"
4. Click Next
5. Name it "Split Screen App"
6. Click Finish

After creating the shortcut:
1. Right-click the shortcut
2. Select Properties
3. In the "Start in" field, paste this path:
   C:\Users\Kieran Jackson\screen_split_app\
4. Click OK

If the shortcut still doesn't work:
1. Open Command Prompt
2. Navigate to the app folder:
   cd /d "C:\Users\Kieran Jackson\screen_split_app\"
3. Run the app directly:
   python screen_app_debug.pyw

This will help identify if there are any errors when running the app.
